Jednoduchá textová hra s názvem "Dungeon game"

Program lze spustit přes přiložený jar, úkolem hráče je navigovat se dungeonem a porazit Big chunguse, který má předmět
Thanosova rukavice, který umožní hráči vyhrát hru.

Jiří Vrba

verze 1.0, programováno během letního semestru 2020/2021
